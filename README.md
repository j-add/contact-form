# Contact Form

A simple contact form

## Requirements

- Form should include fields for:
  - Name
  - Email
  - Comments/Message
- Clean design

## Technologies
- HTML & SASS form design/structure
- JS front-end validation
- PHP back-end sanitisation, validation, and data processing
